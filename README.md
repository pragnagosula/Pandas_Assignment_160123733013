# 🌍 World Happiness Report Analysis

This repository contains a comprehensive exploratory data analysis (EDA) of the World Happiness Report dataset across multiple years (2015–2019). The project focuses on understanding global happiness trends, regional variations, and the impact of socio-economic factors using Python and Pandas.

---

## 📊 Dataset

**Files Used:**

* 2015.csv
* 2016.csv
* 2017.csv
* 2018.csv
* 2019.csv

**Description:**
The dataset contains country-level information on happiness scores along with contributing factors such as GDP per capita, social support, life expectancy, freedom, generosity, and corruption perception.

---

## 🚀 Project Overview

This analysis includes:

* Data cleaning and standardization across multiple years
* Handling missing values using mean, forward fill, and interpolation
* Creating a unified dataset with consistent column names
* Regional analysis of happiness variation using groupby operations
* Correlation analysis between GDP and happiness
* Year-wise comparison using index alignment
* Hierarchical indexing (Region + Country) for multi-level analysis
* Boolean indexing to identify highly happy countries
* Designing a composite **Well-Being Index** using normalized features
* Comparing top 20 and bottom 20 countries using index-aligned operations
* Evaluating the impact of missing data handling on regional rankings
* Multi-level pivot table analysis across regions and years

---

## 🛠️ Key Techniques Used

* Pandas (groupby, pivot_table, indexing, ufuncs)
* Data preprocessing and cleaning
* Boolean indexing and filtering
* Handling missing data (fillna, interpolation)
* Vectorized operations for performance
* Multi-level indexing and aggregation

---

## 📈 Key Insights

* Happiness is influenced by multiple factors, not just GDP
* Social support and life expectancy play a major role
* Regional disparities highlight inequality in living conditions
* Data preprocessing significantly affects analytical outcomes
* Composite indices provide a more balanced understanding of well-being
